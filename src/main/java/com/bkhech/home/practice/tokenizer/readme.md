# 中文分词

## Jieba
- https://www.oschina.net/p/jieba-analysis
- https://github.com/huaban/jieba-analysis.git
```
简介
    支持分词模式
        Search模式，用于对用户查询词分词
        Index模式，用于对索引文档分词
    如何获取
        git clone https://github.com/huaban/jieba-analysis.git
        cd jieba-analysis
        mvn package install ，安装到本地仓库
        在你的 pom.xml 文件中添加以下依赖。
```
